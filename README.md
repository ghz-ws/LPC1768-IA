# Impedance Analyzer

Impedance analyzer based with LPC1768 and AD9834.<br>
Up to 1 MHz.<br>
Python code is a software frontpanel script.<br>
Related book (Japanese) is available from https://ghz-ws.booth.pm/items/5209259 <br>
